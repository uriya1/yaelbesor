# יעל בשור - אתר מקצועי

אתר מודרני עבור יעל בשור - מטפלת זוגית, משקמת יחסים ומגשרת.

## 🚀 טכנולוגיות

- **Next.js 14** - React Framework עם App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - עיצוב מודרני ורספונסיבי
- **Lucide React** - אייקונים יפים

## 📁 מבנה הפרויקט

```
yaelbesor-website/
├── app/                    # Next.js App Router pages
│   ├── page.tsx           # דף הבית
│   ├── layout.tsx         # Layout ראשי
│   ├── globals.css        # סגנונות גלובליים
│   ├── couple-therapy/    # דף טיפול זוגי
│   ├── gishor/            # דף גישור
│   ├── parenthood/        # דף הורות ומשפחה
│   ├── quiet-birth/       # דף תמיכה לאחר לידה שקטה
│   └── faq/               # דף שאלות נפוצות
├── components/            # React Components
│   ├── Header.tsx         # תפריט ניווט
│   ├── Footer.tsx         # פוטר
│   ├── ContactForm.tsx    # טופס יצירת קשר
│   ├── FAQAccordion.tsx   # אקורדיון שאלות נפוצות
│   └── ServicePageLayout.tsx  # Layout לדפי שירותים
├── public/                # קבצים סטטיים
│   └── images/           # תמונות
└── ...
```

## 🎨 עמודים

1. **דף הבית** (`/`) - Hero, אודות, שירותים, טופס יצירת קשר
2. **טיפול זוגי** (`/couple-therapy`) - מידע על טיפול זוגי
3. **גישור** (`/gishor`) - מידע על גישור לזוגות
4. **הורות ומשפחה** (`/parenthood`) - זוגיות אחרי לידה, זוגיות צעירה
5. **תמיכה לאחר לידה שקטה** (`/quiet-birth`) - ליווי ותמיכה
6. **שאלות נפוצות** (`/faq`) - Q&A מקיף

## 🛠️ התקנה והרצה

```bash
# התקנת תלויות
npm install

# הרצה במצב פיתוח
npm run dev

# בנייה לפרודקשן
npm run build

# הרצה בפרודקשן
npm start
```
